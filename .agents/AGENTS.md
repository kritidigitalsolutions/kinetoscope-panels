# Kinetoscope Project Custom Rules

- **Clean Up Debug Files**: Always clean up and delete any temporary, debug, test, or scratch files created during code debugging or analysis (e.g., SQLite databases, JSON logs, temp folders) before completing the task.
- **Revert Temporary Configs**: Keep config files (like `vite.config.js` or `.env`) pristine by reverting any temporary debug middlewares or endpoints after use.
